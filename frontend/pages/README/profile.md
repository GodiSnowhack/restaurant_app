# Страница профиля пользователя (/profile)

## Назначение
Страница для просмотра и редактирования личной информации пользователя. Доступна только авторизованным пользователям.

## Функциональность
- Отображает персональную информацию пользователя (имя, email, телефон)
- Предоставляет возможность редактирования персональных данных
- Позволяет изменить пароль учетной записи
- Отображает историю заказов пользователя с возможностью просмотра деталей
- Показывает историю бронирований пользователя
- Позволяет управлять настройками уведомлений
- Предоставляет возможность добавления/редактирования адресов доставки
- Отображает имеющиеся скидки и бонусы пользователя
- Позволяет управлять привязанными способами оплаты
- Адаптируется под темную/светлую тему сайта
- Имеет мобильную и десктопную версии дизайна
- Включает возможность выхода из системы

## Пользовательские действия
- Просмотр и редактирование персональной информации
- Изменение пароля
- Просмотр истории заказов и бронирований
- Управление адресами доставки
- Настройка уведомлений
- Управление способами оплаты
- Выход из системы

## Связанные компоненты
- Header - верхняя навигационная панель
- Footer - нижняя часть страницы
- Формы для редактирования данных
- Компоненты для отображения истории заказов
- Компоненты для управления адресами и способами оплаты
